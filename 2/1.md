## Variable and Parameter Expansion

**By Using Simple Variable** 
1. Make Variable **employee** with the name **Sarah**
2. Dear **Sarah**
    The Board and I would like to thank you for the excellent work that you have been doing over the last few months
    Keep up the good work, **Sarah**
    Best wishes,
    Karen, CEO
3. Save and run it

**By Using Parameter Expansion**
1. Instead using \$variable change it into ${instead} 
2. Save and run it


<details>
<summary>Solution</summary>
employee="Sarah"
Dear ${employee}
    The Board and I would like to thank you for the excellent work that you have been doing over the last few months
    Keep up the good work, ${employee}
    Best wishes,
    Karen, CEO
</details>
